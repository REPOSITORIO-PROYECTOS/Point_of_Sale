# Frontend (React + Vite)

UI del punto de venta.

## Scripts

```powershell
npm install
npm run dev      # http://localhost:5173
npm run build    # salida en frontend/dist/
```

El proxy `/api` apunta a `http://127.0.0.1:3001` (ver `vite.config.ts`).

Comandos del monorepo (stack completo, desktop, builds): ver [`../README.md`](../README.md).
