# Release v0.0.7

**Fecha:** 2026-06-26  
**Instalador:** [Point-of-Sale-Setup.exe](https://github.com/REPOSITORIO-PROYECTOS/Point_of_Sale/releases/download/v0.0.7/Point-of-Sale-Setup.exe) (~109 MB)

## Resumen

Instalador más liviano y rápido de instalar. Incluye correcciones de sesión/auth de 0.0.6, puertos locales altos (58xxx) y mejoras de caja.

## Cambios principales

### Instalador y runtime

- **NSIS `compression: normal`** — instalación más rápida (descarga ~109 MB vs ~127 MB en 0.0.6).
- **Sin `node.exe` embebido** (~70 MB menos en disco) — el backend usa Electron como Node (`ELECTRON_RUN_AS_NODE`).
- **Puertos empaquetados:** API `58001`, AFIP `58086` (no chocan con 3001/5173 de otras apps).

### App (desde 0.0.6)

- Cierre de sesión automático ante **401** (token expirado).
- Mensajes claros en importación / bulk de productos.
- Auto-actualización con repo público vía `updater.env`.

### Dev / monorepo

- Puertos dev unificados en `config/ports.json`: UI `58173`, API `58001`, AFIP `58086`.

## Actualizar cajas con auto-update

1. En cada PC, una sola vez (si no está hecho):

   ```powershell
   .\scripts\deploy-updater-config.ps1 -Public
   ```

2. Abrir Point of Sale → footer **Buscar actualización** → instalar 0.0.7.

## Instalación manual

```powershell
gh release download v0.0.7 -p Point-of-Sale-Setup.exe -O "$env:TEMP\Point-of-Sale-Setup.exe"
Start-Process "$env:TEMP\Point-of-Sale-Setup.exe"
```

Cerrar todas las ventanas del POS antes de instalar. Tras instalar: **iniciar sesión de nuevo**.

## Verificación

```powershell
Invoke-RestMethod http://127.0.0.1:58001/api
```

## Rollback

Reinstalar [v0.0.6](https://github.com/REPOSITORIO-PROYECTOS/Point_of_Sale/releases/tag/v0.0.6) si hiciera falta (usa puertos 3001/5086).
