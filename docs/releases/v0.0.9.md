# Release v0.0.9

**Fecha:** 2026-06-26  
**Instalador:** [Point-of-Sale-Setup.exe](https://github.com/REPOSITORIO-PROYECTOS/Point_of_Sale/releases/download/v0.0.9/Point-of-Sale-Setup.exe)

## Resumen

Impresión de **cierre de caja / lote** desde auditoría, reutilizando la misma tubería de tickets (vista previa, térmica y PDF). CI publica el release automáticamente al terminar el build.

## Cambios principales

### Cierre de caja

- Botón **Imprimir** en el detalle de cierre (`ClosingDetailModal`).
- Plantilla de ticket **cierre de lote** (HTML y texto térmico) con totales por medio de pago, movimientos y listado de ventas.
- Módulo `cash-closing-receipt.ts` y extensión de `print-receipt.ts` para el nuevo tipo de comprobante.

### CI / release

- El workflow `release-desktop.yml` quita el draft y marca `latest` al finalizar el build (evita releases publicados sin assets).

## Actualizar cajas

1. Footer **Buscar actualización** → instalar 0.0.9, o:
2. Manual:

```powershell
gh release download v0.0.9 -p Point-of-Sale-Setup.exe -O "$env:TEMP\Point-of-Sale-Setup.exe"
Start-Process "$env:TEMP\Point-of-Sale-Setup.exe"
```

Cerrar Point of Sale antes de instalar.

## Verificación

```powershell
Invoke-RestMethod http://127.0.0.1:58001/api
```

En Auditoría → abrir un cierre → **Imprimir** / vista previa del ticket de lote.
