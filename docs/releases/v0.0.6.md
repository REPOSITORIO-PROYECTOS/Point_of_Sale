# Release v0.0.6

**Fecha:** 2026-06-26  
**Instalador:** [Point-of-Sale-Setup.exe](https://github.com/REPOSITORIO-PROYECTOS/Point_of_Sale/releases/download/v0.0.6/Point-of-Sale-Setup.exe) (~127 MB)

## Cambios

- **Auth:** cierre de sesión automático ante HTTP 401 (token expirado).
- **Import/bulk:** mensajes de error más claros cuando falta sesión.
- **Permisos:** `canEditProducts` / `canViewAudit` en UI.
- **Auto-update:** soporte repo público (`POS_UPDATER_PUBLIC_REPO`) y `updater.env` en AppData.

## Actualizar

```powershell
.\scripts\deploy-updater-config.ps1 -Public
```

Luego en el POS: footer → **Buscar actualización**.

> **Recomendado:** usar [v0.0.7](v0.0.7.md) (instalador más liviano y puertos 58xxx).
